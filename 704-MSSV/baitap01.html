Bài 1 - Image Carousel

Carousel hình ảnh với CSS transition, auto-play 3 giây, next/prev và responsive

Giải thích logic tư duy

1. Cách xử lý index để tránh ngoài giới hạn

Sử dụng phép toán modulo (%) để đảm bảo index luôn nằm trong phạm vi hợp lệ:

  • currentIndex = ((index % totalSlides) + totalSlides) % totalSlides
  • Khi nhấn Next ở slide cuối (index = 5): (5 + 1) % 6 = 0 → quay về slide đầu
  • Khi nhấn Prev ở slide đầu (index = 0): (0 - 1 + 6) % 6 = 5 → nhảy đến slide cuối
  • Phép toán ((n % m) + m) % m xử lý cả trường hợp index âm (JavaScript modulo có thể trả về số âm)

2. Tối ưu performance

  • CSS Transform thay vì thay đổi layout: Sử dụng transform: translateX() được GPU tăng tốc, không gây reflow/repaint toàn trang
  • will-change: transform: Báo trước cho browser chuẩn bị tối ưu composite layer
  • CSS transition thay vì JS animation: Browser tối ưu CSS transition tốt hơn, chạy trên compositor thread riêng
  • Pause khi hover: Dừng setInterval khi user tương tác giúp tiết kiệm CPU
  • Event passive: Touch events sử dụng { passive: true } để không block scrolling
